### 🚀 功能模块

| 模块 | 图标 | 功能描述 | 现代化特性 |

## 📖 使用指南

### 1. 首次访问
1. 启动服务后访问 http://localhost:7860
2. 查看Landing页面了解功能特性
3. 点击"创作工坊"开始使用

### 2. 配置设置
1. 访问"模型配置"标签页
2. 设置LLM和Embedding参数
3. 使用测试功能验证配置
4. 查看右侧日志获取详细反馈

### 3. 创作流程
1. 在"小说参数"设置基本信息
2. 在"创作工坊"按Step1-4流程操作
3. 使用"章节管理"编辑内容
4. 通过"角色库"管理角色信息
